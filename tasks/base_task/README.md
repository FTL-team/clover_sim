# Base clover task


This is base task for cloversim that provides world with 10 by 10 meters aruco map. 

This task also contains simple checker, that tests flying in square with side of 1.
Exact steps drone should execute:
* Takeoff
* Fly 1m positive x
* Fly 1m positive y
* Fly 1m negative x
* Fly 1m negative y
* Land
